# Real Estate Starter (React + Node + PostgreSQL + Docker)

## Quick start (requires Docker & docker-compose)

1. Build and run:
```
docker-compose up --build
```

2. Backend: http://localhost:5000
   Frontend: http://localhost:3000

3. Create a user via POST /api/users/register or use the frontend (you may need to add register page).

Notes:
- This is a development starter. Move uploads to S3 for production and secure environment variables.
